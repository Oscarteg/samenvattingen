# Abstract Factory
```
Je zegt dat je ook gebruik maakt van Repository Pattern. Dit is geen Pattern die behandeld wordt. Als je zegt dat je abstract factory method behandeld: dan graag ook het algemene plaatje van dit pattern en hoe deze is toegepast. Nu noem je alleen dependency injection, dit is eigenlijk ook geen pattern is, maar iets wat het pattern gebruikt. Verder vind ik de indeling iets verwarrend. bijvoorbeeld Object Interactie, figuur 5 staat weer achteraan.
``` 
